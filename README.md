# Weather Application

## Description
A responsive weather application developed using HTML, CSS, and JavaScript. The application fetches real-time weather information using a Weather API and displays temperature, humidity, weather conditions, and wind speed for any searched city.

## Features
- Search weather by city
- Real-time weather updates
- Temperature and humidity
- Wind speed
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## How to Run
1. Download or clone the repository.
2. Open index.html in your browser.
